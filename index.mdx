## Acceso

Para crear tu cuenta, visita: https://app.getsupervisor.ai

Para entrar a Agents Studio (con la misma cuenta): https://agents.studio.getsupervisor.ai

## Qué puedes hacer hoy

- Crear y gestionar agentes con versioning, estados y Playground
- Diseñar conversaciones con el Flow Editor (drag & drop)
- Conectar una Knowledge Base (documentos con vectorización)
- Activar Tools (voz/telefonía, geocoding, human handoff)
- Operar en Sandbox/Production dentro de tu Workspace

## Tools e integraciones

- Voice/Telephony Tool para experiencias de voz
- Geocoding Tool para ubicaciones y distancias
- Human Handoff Tool para escalar a un humano
- Knowledge Base connector para indexar documentos (Drive‑like)
- Messaging channels (p. ej., WhatsApp, web chat)
