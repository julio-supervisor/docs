## Autenticación

Todos los endpoints requieren una API Key simple (estilo n8n).

- Header: `X-API-Key: <TU_API_KEY>`
- Cómo obtenerla: desde la propia herramienta, copia tu clave en la sección de API Keys.

Ejemplo (pseudocódigo):

```http
POST /webhook/... HTTP/1.1
Host: api.workflows.getsupervisor.ai
Content-Type: application/json
X-API-Key: TU_API_KEY

{ "chatbotId": "...", "phone": "..." }
```
